Quickstart:

Suppose you want pieces with flute that plays in range D4 G#6 and viola in C3 E5...

  1. Enter an instrument range query in the search box below as "flute D4 G#6 and viola C3 E5" (without quotes)
  2. When you've entered all your queries, hit the "Toscanini Search" button and we'll show you our matching scores as PDF or MusicXML
More Queries
  • -See all scores we have by leaving the search box empty
  • -Query for a single composer, ex: "bach"
  • -Query for tempo between range, ex: "tempo 40 130", however if a piece has no written tempo it will pass for any tempo
  • -Query for a single key signature (if a piece has this key and or others it will pass), ex: "key Bb"
  • -Here's a complex example query: "Beethoven and tempo 80 170 and flute D4 G#6 and trumpet F#3 A5 and key Bb"
Side Notes
  • -This engine is not case sensitive
  • -Queries check for name inclusion, so querying for "trumpet 40 70" can give you scores with "Trumpets in C" assuming the range is appropriate
